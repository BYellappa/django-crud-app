# Django CRUD Project

This is a simple CRUD (Create, Read, Update, Delete) web application built using Django.

📂 Project Structure
fbvcrudproject/
│
├── testapp/
│   ├── models.py
│   ├── views.py
│   ├── forms.py
│   ├── urls.py
│
├── templates/
│   ├── base.html
│   ├── index.html
│   ├── insert.html
│   ├── update.html
│
├── db.sqlite3
├── manage.py

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## Technologies Used
- Python
- Django
- HTML

## How to Run
1. Clone the repo
2. Run `python manage.py runserver`
3. Open http://127.0.0.1:8000/

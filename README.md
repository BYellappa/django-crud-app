# Django CRUD Project

This is a simple CRUD (Create, Read, Update, Delete) web application built using Django.

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## Technologies Used
- Python
- Django
- HTML

## How to Run
1. Clone the repo
2. Run `python manage.py runserver`
3. Open http://127.0.0.1:8000/
